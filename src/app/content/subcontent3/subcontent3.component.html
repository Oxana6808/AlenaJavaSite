3 Условия boolean переменная и вывод значения

Условия позволяют выполнять различные действия в зависимости от значения переменной. В этом разделе мы рассмотрим проверку условий с использованием операторов сравнения и булевых переменных.

Пример кода с инициализацией boolean переменной:

public class BooleanConditionExample {
    public static void main(String[] args) {
        // Инициализируем булевую переменную на основе условия 5 > 2
        boolean isGreater = 5 > 2;

        // Выводим значение переменной
        System.out.println("5 больше 2: " + isGreater);
    }
}

Вывод программы:

5 больше 2: true
