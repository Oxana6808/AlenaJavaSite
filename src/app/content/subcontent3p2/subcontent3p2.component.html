3-2 Выводим большее число из 2

Задача выводит большее из двух чисел с использованием условий.

Пример кода:

public class MaxOfTwoExample {
    public static void main(String[] args) {
        int a = 10;
        int b = 20;

        if (a > b) {
            System.out.println("Большее число: " + a);
        } else {
            System.out.println("Большее число: " + b);
        }
    }
}

Вывод программы:

Большее число: 20
