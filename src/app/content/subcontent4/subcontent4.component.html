4 Цикл вывод 10 *

Циклы позволяют повторять блок кода несколько раз. В этом разделе мы рассмотрим различные программы с использованием циклов для выполнения повторяющихся операций, таких как вывод чисел или работа с массивами.

Пример кода для вывода 10 звездочек (*):

public class StarPrintExample {
    public static void main(String[] args) {
        for (int i = 0; i < 10; i++) {
            System.out.print("*");
        }
    }
}
