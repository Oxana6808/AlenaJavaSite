2-1 Задача на унарные операторы -x +x

Унарные операторы -x и +x позволяют изменять знак числа. Оператор +x не изменяет значение числа, тогда как -x изменяет его на противоположное.

Пример кода:

public class UnaryOperatorsExample {
    public static void main(String[] args) {
        int x = 5;
        System.out.println("x: " + x);
        System.out.println("-x: " + (-x));
        System.out.println("+x: " + (+x));
    }
}

Вывод программы:

x: 5
-x: -5
+x: 5
