4-6 Создать программу, которая находит сумму всех элементов массива

Эта программа вычисляет сумму всех элементов массива.

Пример кода:

public class ArraySumExample {
    public static void main(String[] args) {
        int[] array = {1, 2, 3, 4, 5};
        int sum = 0;
        for (int i = 0; i < array.length; i++) {
            sum += array[i];
        }
        System.out.println("Сумма элементов массива: " + sum);
    }
}

Вывод программы:

Сумма элементов массива: 15
