3-3 Выводим большее из 3

В этой задаче мы определим наибольшее число из трех введенных чисел.

Пример кода:

public class MaxOfThreeExample {
    public static void main(String[] args) {
        int a = 5;
        int b = 12;
        int c = 9;

        if (a >= b && a >= c) {
            System.out.println("Наибольшее число: " + a);
        } else if (b >= a && b >= c) {
            System.out.println("Наибольшее число: " + b);
        } else {
            System.out.println("Наибольшее число: " + c);
        }
    }
}

Вывод программы:

Наибольшее число: 12
