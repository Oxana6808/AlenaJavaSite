3-6 Выводим используя else if

Программа проверяет, какие задачи были выполнены, и выводит соответствующие сообщения. Если ни одна задача не выполнена, программа выводит "учимся".

Пример кода:

public class TasksExample {
    public static void main(String[] args) {
        boolean didSchulte = false;
        boolean didProgramming = true;
        boolean didShowEnglish = false;
        boolean didCounting = false;
        boolean didReading = false;

        if (didSchulte) {
            System.out.println("Сделала шульте");
        } else if (didProgramming) {
            System.out.println("Сделала программирование");
        } else if (didShowEnglish) {
            System.out.println("Сделала шоу инглиш");
        } else if (didCounting) {
            System.out.println("Сделала счет");
        } else if (didReading) {
            System.out.println("Почитала");
        } else {
            System.out.println("Учимся");
        }
    }
}
