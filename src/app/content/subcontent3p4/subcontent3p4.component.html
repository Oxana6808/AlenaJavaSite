3-4 Выводим текст "Идем на море", если солнечно (true)

Если переменная sunny равна true, программа выводит "Идем на море".

Пример кода:

public class SunnyDayExample {
    public static void main(String[] args) {
        boolean sunny = true;

        if (sunny) {
            System.out.println("Идем на море");
        } else {
            System.out.println("Остаемся дома");
        }
    }
}

Вывод программы:

Идем на море
