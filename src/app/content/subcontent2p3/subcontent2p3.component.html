2-3 Задача на проверку равенства или неравенства

Операторы == и != используются для проверки равенства и неравенства двух значений соответственно. Результатом этих операций является булево значение (true или false).

Пример кода:

public class EqualityCheckExample {
    public static void main(String[] args) {
        int a = 5;
        int b = 10;
        System.out.println("a == b: " + (a == b));
        System.out.println("a != b: " + (a != b));
    }
}

Вывод программы:

a == b: false
a != b: true
