3-7 Используем конструкцию switch case при переходе дороги

Программа использует оператор switch case для описания действий при разных сигналах светофора.

Пример кода:

public class TrafficLightExample {
    public static void main(String[] args) {
        String light = "красный";

        switch (light) {
            case "красный":
                System.out.println("Стоим");
                break;
            case "зеленый":
                System.out.println("Идем");
                break;
            default:
                System.out.println("Ждем");
                break;
        }
    }
}

Вывод программы:

Стоим
