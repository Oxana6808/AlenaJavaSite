2-2 Задача на оператор +=, -=, *=

Операторы +=, -=, *= используются для сокращённой записи операций присваивания. Например, a += b означает a = a + b.

Пример кода:

public class AssignmentOperatorsExample {
    public static void main(String[] args) {
        int a = 5;
        a += 3; // a = a + 3
        System.out.println("После a += 3: " + a);
        a -= 2; // a = a - 2
        System.out.println("После a -= 2: " + a);
        a *= 4; // a = a * 4
        System.out.println("После a *= 4: " + a);
    }
}

Вывод программы:

После a += 3: 8
После a -= 2: 6
После a *= 4: 24
