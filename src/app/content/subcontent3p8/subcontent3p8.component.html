3-8 Задача на тернарный оператор

Программа использует тернарный оператор для вывода сообщения в зависимости от того, является ли число четным или нечетным.

Пример кода:

public class TernaryExample {
    public static void main(String[] args) {
        int number = 7;

        // Используем тернарный оператор для проверки четности числа
        String result = (number % 2 == 0) ? "Число четное" : "Число нечетное";

        // Выводим результат
        System.out.println(result);
    }
}

Вывод программы:

Число нечетное
