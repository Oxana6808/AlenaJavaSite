4-9 Написать программу, которая находит число 5 в массиве и выводит его индекс

Эта программа находит число 5 в массиве и выводит его индекс.

Пример кода:

public class FindIndexExample {
    public static void main(String[] args) {
        int[] array = {1, 5, 3, 7, 5};
        for (int i = 0; i < array.length; i++) {
            if (array[i] == 5) {
                System.out.println("Число 5 найдено на индексе: " + i);
            }
        }
    }
}

Вывод программы:

Число 5 найдено на индексе: 1
Число 5 найдено на индексе: 4
