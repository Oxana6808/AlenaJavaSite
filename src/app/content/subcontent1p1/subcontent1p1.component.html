1-1 Вывод на консоль println

В Java для вывода текста на консоль используются функции print и println. println добавляет символ новой строки после вывода текста, тогда как print - нет. Это означает, что вывод, следующий за println, начнется с новой строки, в то время как вывод после print продолжится на той же строке.

Пример ввода и вывода:

// Пример использования функций print и println
public class PrintExample {
    // Декларируем главный метод main
    public static void main(String[] args) {
        System.out.print("Hello");
        System.out.print(", ");
        System.out.print("World");
        System.out.println("!");
        System.out.println("Это новая строка.");
    }
}

В результате выполнения этой программы на консоль будет выведено:

Hello, World!
Это новая строка.

Метод println() добавляет символ новой строки после вывода текста, в отличие от print(). Это означает, что каждая новая строка начинается с новой линии.

Пример кода:

public class PrintlnExample {
    public static void main(String[] args) {
        System.out.println("Hello");
        System.out.println("World!");
    }
}

Вывод программы:

Hello
World!
