1-1 Выводим в одну строку "Hello, world!"

Написать программу, которая выводит на экран текст "Hello world!".

Решение:


// Декларируем класс HelloWorld
public class HelloWorld {
    // Декларируем главный метод main
    public static void main(String[] args) {
        // Выводим на экран строку "Hello world!" с помощью метода println
        System.out.print("Hello, world!");
    }
}

В результате выполнения этой программы на консоль будет выведено:


Hello, world!

Задания для самостоятельной работы:

  • Вывести на экран в одну строку:
    • "Я изучаю язык программирования - Java!"
    • "Алёна любит учиться!"
    • "Java один из самых популярных языков программирования!"
    • "Java основан 23 мая 1995г. Джеймсом Гослингом в компании Sun Microsystems."
    • "Компания Oracle выкупила права на Java."
