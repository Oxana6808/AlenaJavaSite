2-4 Задача на логические операторы >>, <<, ~

Операторы сдвига >> , << и побитовая инверсия ~ выполняют побитовые операции. Сдвиг вправо и влево изменяет положение битов числа, а побитовая инверсия изменяет все биты числа на противоположные.

Пример кода:

public class BitwiseOperatorsExample {
    public static void main(String[] args) {
        int a = 8; // 1000 в двоичном виде
        System.out.println("a >> 1: " + (a >> 1)); // сдвиг вправо
        System.out.println("a << 1: " + (a << 1)); // сдвиг влево
        System.out.println("~a: " + (~a)); // побитовая инверсия
    }
}

Вывод программы:

a >> 1: 4
a << 1: 16
~a: -9
