4-10 Написать программу, которая выводит таблицу итераций при обходе массива, с подсветкой итераций

Эта программа выводит таблицу, в которой отображаются итерации при обходе массива.

Пример кода:

public class IterationTableExample {
    public static void main(String[] args) {
        int[] array = {1, 2, 3, 4, 5};
        System.out.println("Итерация\tЗначение");
        for (int i = 0; i < array.length; i++) {
            System.out.println(i + "\t\t" + array[i]);
        }
    }
}

Вывод программы:

Итерация    Значение
0           1
1           2
2           3
3           4
4           5
