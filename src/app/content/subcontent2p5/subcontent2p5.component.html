2-5 Задача на ++ и --

Операторы инкремента ++ и декремента -- увеличивают или уменьшают значение переменной на 1 соответственно. Эти операторы могут быть использованы как в префиксной (++x, --x), так и постфиксной форме (x++, x--).

Пример кода:

public class IncrementDecrementExample {
    public static void main(String[] args) {
        int x = 5;
        System.out.println("x до ++: " + x);
        System.out.println("x++: " + (x++));
        System.out.println("x после ++: " + x);
        System.out.println("x до --: " + x);
        System.out.println("x--: " + (x--));
        System.out.println("x после --: " + x);
    }
}

Вывод программы:

x до ++: 5
x++: 5
x после ++: 6
x до --: 6
x--: 6
x после --: 5
