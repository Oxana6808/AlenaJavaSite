1-3 Выводим красным цветом "Hello, " и синим "world!"

Для подсвечивания цвета слов, фона разными цветами, жирного шрифта и сброса есть ANSI-коды:

Цвета текста:

  • красный - "\u001B[31m"
  • желтый - "\u001B[33m"
  • зеленый - "\u001B[32m"
  • синий - "\u001B[34m"
  • белый - "\u001B[37m"

Цвета фона:

  • красный фон - "\u001B[41m"
  • желтый фон - "\u001B[43m"
  • зеленый фон - "\u001B[42m"
  • синий фон - "\u001B[44m"
  • белый фон - "\u001B[47m"
  • сброс цвета - "\u001B[0m"
  • жирный шрифт - "\u001B[1m"

Написать программу, которая выводит на консоль красным цветом "Hello, " и синим "world!":

// Пример использования ANSI-кодов для цветного вывода
public class ColorExample {
    public static void main(String[] args) {
        // ANSI-код для сброса цвета
        final String RESET = "\u001B[0m";
        // ANSI-коды для цветов
        final String RED = "\u001B[31m";
        final String BLUE = "\u001B[34m";

        // Вывод текста с использованием цветов
        System.out.print(RED + "Hello, " + RESET);
        System.out.print(BLUE + "world!" + RESET);
    }
}

В результате выполнения этой программы на консоль будет выведено:

Hello, world!

Задания для самостоятельной работы:

  • Написать красным "Я изучаю язык программирования - Java!"
  • Написать желтым "Алёна любит учиться!"
  • Написать зеленым "Java один из самых популярных языков программирования!"
  • Написать на синем фоне "Java основан 23 мая 1995г. Джеймсом Гослингом в компании Sun Microsystems."
  • Написать на желтом фоне красным цветом Oracle выкупила права на Java."
