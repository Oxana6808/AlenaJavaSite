1 Вывод на консоль print

В Java для вывода текста на консоль используется функция print. В отличие от println, метод print не добавляет символ новой строки. Это означает, что следующий вывод продолжается на той же строке.

Пример кода:

public class PrintExample {
    public static void main(String[] args) {
        System.out.print("Hello");
        System.out.print(", ");
        System.out.print("World!");
    }
}

Вывод программы:

Hello, World!
