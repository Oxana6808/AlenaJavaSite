4-11 Написать программу, которая подсвечивает значения при сортировке массива по возрастанию. (Пузырьковая сортировка)

Эта программа использует пузырьковую сортировку для сортировки массива и отображает каждый шаг сортировки.

Пример кода:

public class BubbleSortExample {
    public static void main(String[] args) {
        int[] array = {5, 2, 9, 1, 5, 6};
        boolean swapped;
        do {
            swapped = false;
            for (int i = 0; i < array.length - 1; i++) {
                if (array[i] > array[i + 1]) {
                    int temp = array[i];
                    array[i] = array[i + 1];
                    array[i + 1] = temp;
                    swapped = true;
                    // Выводим шаг сортировки
                    System.out.println("Шаг сортировки: " + java.util.Arrays.toString(array));
                }
            }
        } while (swapped);
    }
}

Вывод программы:

Шаг сортировки: [2, 5, 1, 5, 6, 9]
Шаг сортировки: [2, 1, 5, 5, 6, 9]
Шаг сортировки: [1, 2, 5, 5, 6, 9]
