3-1 Проверяем больше число 5, отрицательное или положительное, или проверка на равенство

В этой задаче мы проверяем число: больше ли оно 5, является ли оно положительным или отрицательным, и равняется ли оно 5.

Пример кода:

public class NumberCheckExample {
    public static void main(String[] args) {
        int number = 5;

        if (number > 5) {
            System.out.println("Число больше 5");
        } else if (number < 0) {
            System.out.println("Число отрицательное");
        } else if (number == 5) {
            System.out.println("Число равно 5");
        } else {
            System.out.println("Число меньше 5");
        }
    }
}

Вывод программы:

Число равно 5
