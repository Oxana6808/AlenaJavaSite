4-4 Создать программу, которая вычисляет сумму чисел от 1 до 100

Эта программа использует цикл для вычисления суммы чисел от 1 до 100.

Пример кода:

public class SumExample {
    public static void main(String[] args) {
        int sum = 0;
        for (int i = 1; i <= 100; i++) {
            sum += i;
        }
        System.out.println("Сумма: " + sum);
    }
}

Вывод программы:

Сумма: 5050
