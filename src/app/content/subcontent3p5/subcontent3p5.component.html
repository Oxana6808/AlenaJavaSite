3-5 Проверяем является ли число четным

Программа проверяет, является ли число четным, используя оператор % для вычисления остатка от деления.

Пример кода:

public class EvenCheckExample {
    public static void main(String[] args) {
        int number = 8;

        if (number % 2 == 0) {
            System.out.println("Число четное");
        } else {
            System.out.println("Число нечетное");
        }
    }
}

Вывод программы:

Число четное
