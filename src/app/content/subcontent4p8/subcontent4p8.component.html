4-8 Создать программу, которая находит наименьшее число в массиве

Эта программа находит наименьшее число в массиве.

Пример кода:

public class MinInArrayExample {
    public static void main(String[] args) {
        int[] array = {3, 9, 7, 1, 5};
        int min = array[0];
        for (int i = 1; i < array.length; i++) {
            if (array[i] < min) {
                min = array[i];
            }
        }
        System.out.println("Наименьшее число: " + min);
    }
}

Вывод программы:

Наименьшее число: 1
