1-2 Экранированные символы

В Java для вывода специальных символов, таких как кавычки или обратный слэш, используются экранированные последовательности, например \" для кавычек и \\ для обратного слэша.

Пример кода:

public class EscapeCharactersExample {
    public static void main(String[] args) {
        System.out.println("Это \"цитата\"");
        System.out.println("Путь к файлу: C:\\\\Users\\\\Documents");
    }
}

Вывод программы:

Это "цитата"
Путь к файлу: C:\Users\Documents
