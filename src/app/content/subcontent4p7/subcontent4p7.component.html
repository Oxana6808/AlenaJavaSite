4-7 Написать программу, которая находит наибольшее число в массиве

Эта программа находит наибольшее число в массиве.

Пример кода:

public class MaxInArrayExample {
    public static void main(String[] args) {
        int[] array = {3, 9, 7, 1, 5};
        int max = array[0];
        for (int i = 1; i < array.length; i++) {
            if (array[i] > max) {
                max = array[i];
            }
        }
        System.out.println("Наибольшее число: " + max);
    }
}

Вывод программы:

Наибольшее число: 9
